# Security Policy

This project does not include any secrets, API keys, or user credentials in source control. For security and privacy:

- Do **not** commit any personal keys, tokens, or configuration files containing secrets.
- Keep sensitive data (if any) outside the repository and use environment variables or external configuration tools.
- The mod does not transmit user data or connect to external services.

## Reporting security issues

If you find a vulnerability or security issue, please open an issue in the repository and mark it as a security report. Include steps to reproduce, affected versions, and any relevant logs.
